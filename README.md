# todo-haskell
A small *to-do* app for practice

MVP Goals
------------
* Create a text todo with a simple signature

```ruby
{ todo: {id: UUID, task: Todo, createdAt: UTCTime} }
```

* Be able to do some simple CRUD, probably just save and retrieve a todo.
